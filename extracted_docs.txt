📘 CAHIER DES CHARGES FONCTIONNEL & TECHNIQUE
Système de gestion de location de voitures
1. 🎯 CONTEXTE DU PROJET
1.1 Présentation
Le projet consiste à développer une solution digitale complète pour une agence de location de voitures, incluant :
un site web de réservation
un backoffice de gestion
une application mobile pour les agents
1.2 Objectifs
Objectifs métier
centraliser la gestion des réservations
automatiser les opérations
réduire les erreurs humaines
améliorer la gestion financière
permettre collaboration avec d’autres agences
Objectifs techniques
architecture scalable
API centralisée
synchronisation temps réel
2. 🧱 ARCHITECTURE TECHNIQUE
Stack
Backend : Laravel
Frontend : Next.js
Mobile : Android (Flutter recommandé)
Base de données : PostgreSQL
Principe
toutes les opérations passent par l’API
logique métier côté backend uniquement
