# VectoriaRentCar - Mobile App

This directory contains the Flutter application for VectoriaRentCar.

## Architecture
- **Framework**: Flutter
- **State Management**: BLoC / Provider
- **Local Storage**: Hive / SQLite (for offline sync)

## Features
- Agent Authentication
- Scan CIN / Driving License (OCR)
- Offline Reservation Creation
- Digital Signature

## Setup
To run this project, you need the Flutter SDK installed.

1. Install [Flutter](https://docs.flutter.dev/get-started/install)
2. Run `flutter pub get`
3. Connect an emulator or physical device.
4. Run `flutter run`
